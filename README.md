
# 🎨 Color Detection Project

Welcome to the **Color Detection Project**! This project empowers you to upload any image, click on any part of it, and instantly identify the color at that spot. The detected color's name, RGB values, and a visual swatch are displayed in real-time. Perfect for designers, developers, and anyone curious about colors!

## 🌟 Features

- **🌐 Upload Image**: Seamlessly upload images from your device.
- **🔍 Real-Time Color Detection**: Click on any point in the image to see the color name and RGB values instantly.
- **🎨 Visual Color Representation**: See the detected color displayed as a visual swatch.
- **🔒 User Authentication**: Secure sign-up, login, and logout functionalities.
- **📱 Responsive Design**: Beautifully responsive interface built with Bootstrap.

## 🛠️ Technologies Used

- **Python** 🐍: The core programming language driving the logic.
- **Flask** 🌐: Lightweight and powerful web framework.
- **OpenCV** 📸: Advanced image processing and color detection.
- **Bootstrap** 💻: Sleek, intuitive, and powerful front-end framework.
- **HTML5 & CSS3** 🎨: Modern standards for structuring and styling the web.

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- Python 3.7+ 🐍
- pip (Python package installer) 📦
- Virtual environment (optional but recommended) 🌐

### Installation Guide

1. **Clone the Repository** 🛠️
    ```bash
    git clone https://github.com/your-username/color-detection-project.git
    cd color-detection-project
    ```

2. **Set Up a Virtual Environment** (Optional but Recommended) 🌐
    ```bash
    python -m venv .venv
    source .venv/bin/activate  # On Windows: .venv\Scripts\activate
    ```

3. **Install Dependencies** 📦
    ```bash
    pip install -r requirements.txt
    ```

4. **Prepare the Assets** 🎨
    - Ensure `color_detection.py`, `colorpic.jpg`, and `colors.csv` are in the project directory.

5. **Run the Application** 🚀
    ```bash
    python app.py
    ```

6. **Access the Web App** 🌐
    - Open your favorite browser and navigate to `http://127.0.0.1:5000/`.

## 🎮 How to Use

1. **Sign Up** 🔑: Create your account to start detecting colors.
2. **Log In** 🔓: Access the color detection feature using your credentials.
3. **Upload Image** 🖼️: Use the "Choose File" button to upload an image.
4. **Detect Colors** 🎯: Click on the image to see the detected color name, RGB values, and a visual swatch.
5. **Log Out** 🚪: Securely log out after your session.
